Política de Privacidade do SmartMeal

Última atualização: 15 de Setembro de 2023

1. Introdução

Esta Política de Privacidade explica como o SmartMeal recolhe, utiliza e protege os seus dados pessoais quando utiliza a nossa Aplicação.

2. Dados que Recolhemos

2.1 Informações da Conta: Nome, endereço de e-mail, senha.

2.2 Dados do Perfil: Preferências alimentares, alergias, tamanho do agregado familiar.

2.3 Dados de Utilização: Seleções de refeições, interações com as listas de compras, dados analíticos.

2.4 Dados do Dispositivo: Modelo do dispositivo, versão do sistema operativo, versão da aplicação, endereço IP.

3. Como Utilizamos os Seus Dados

Para fornecer e personalizar os Serviços, gerar planos de refeições, enviar notificações, melhorar a Aplicação e para análises internas.

4. Partilha e Divulgação

Não vendemos os seus dados pessoais. Podemos partilhar dados com:

  • Fornecedores de Serviços: Alojamento, análises, entrega de e-mail.
  • Conformidade Legal: Se exigido por lei ou para proteger os direitos do SmartMeal.

5. Retenção de Dados

Mantemos os seus dados enquanto a sua conta estiver ativa ou conforme necessário para fornecer os Serviços, e conforme exigido por lei.

6. Segurança

Implementamos medidas de segurança razoáveis (encriptação TLS, servidores seguros) para proteger os seus dados. No entanto, nenhum método é 100% seguro.

7. Os Seus Direitos

Acesso e Portabilidade: Pode solicitar uma cópia dos seus dados.

Retificação: Pode corrigir dados imprecisos.

Eliminação: Pode solicitar a eliminação da sua conta e dados pessoais.

Retirada de Consentimento: Pode retirar o consentimento para o processamento de dados, sujeito a requisitos legais.

8. Privacidade das Crianças

A Aplicação não se destina a crianças com menos de 16 anos. Não recolhemos intencionalmente dados de menores.

9. Links de Terceiros

A Aplicação pode conter links para sites de terceiros. Não somos responsáveis pelas suas práticas de privacidade.

10. Alterações na Política de Privacidade

Podemos atualizar esta Política. Iremos notificá-lo sobre alterações materiais e publicar a data atualizada.

11. Informações de Contacto

Para questões de privacidade, contacte privacy@smartmeal.app.
